# Bundled fonts

Latin Modern (`lmroman10-*`, `lmsans10-*`), the OpenType build of Donald
Knuth's Computer Modern maintained by GUST. These are the fonts the LaTeX
`moderncv` class renders with; the generated CV PDF uses them so it matches
the original LaTeX CV.

Licensed under the GUST Font License (GFL), a free license compatible with
redistribution. See http://www.gust.org.pl/projects/e-foundry/lm/ and the
GFL text at http://www.gust.org.pl/projects/e-foundry/licenses.

Only the faces the CV needs are committed (regular/bold/italic/bolditalic
Roman, regular/bold Sans) to keep the repository small.
